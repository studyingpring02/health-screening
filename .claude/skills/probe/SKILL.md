# /probe

Owner와 공유된 이해에 도달할 때까지 집요하게 조사하라.
회색지대를 없애야한다.

결정들을 하나씩 짚어 내려가며, 서로 얽힌 의존 관계를 풀어라.

Owner의 인지부하를 줄이기 위해,
AskUserQuestion을 사용하여 한 번에 하나씩 질문하라.
각 질문마다 네가 추천하는 답안도 표시하라.

만약 어떤 질문이 현재 디렉토리를 탐색해서 답할 수 있다면, 대신 디렉토리를 탐색하라.
